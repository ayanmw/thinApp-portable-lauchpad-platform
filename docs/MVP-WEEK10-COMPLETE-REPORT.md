## 5. 已知限制（MVP 阶段未解决的问题）

### 5.1 高优先级问题（必须修复）

| 问题 ID | 模块 | 问题描述 | 影响 |
|----------|------|----------|------|
| ISSUE-001 | Hook 引擎 | `ReloadHook()` 可能阻塞调用线程 | UI 卡顿 |
| ISSUE-002 | Hook 引擎 | `printf()` 导致性能下降 | 文件操作延迟增加 10-100 倍 |
| ISSUE-003 | VFS 缓存 | `Get()` 中锁升级可能导致死锁 | 程序卡死 |
| ISSUE-004 | 应用捕获工具 | `CreateSnapshot()` 枚举文件系统太慢 | 捕获启动超时 |

### 5.2 中优先级问题（建议修复）

| 问题 ID | 模块 | 问题描述 | 影响 |
|----------|------|----------|------|
| ISSUE-005 | Hook 引擎 | `original_create_file_w_` 未检查空指针 | 程序崩溃（低概率） |
| ISSUE-006 | Hook 引擎 | `capture_records_` 可能占用大量内存 | 内存占用过高 |
| ISSUE-007 | VFS 缓存 | `IsExpired()` 未考虑 TTL = 0 | 缓存一致性错误 |
| ISSUE-008 | VFS 缓存 | `RemoveTail()` 未检查空指针 | 程序崩溃 |
| ISSUE-009 | 性能监控 | `GetPercentile()` 复制整个数组 | 内存占用高，性能差 |
| ISSUE-010 | 性能监控 | `Record()` 可能重复创建实例 | 内存泄漏（低概率） |
| ISSUE-011 | Launchpad UI | `loadAppList()` 未释放 `QProcess*` | 内存泄漏 |
| ISSUE-012 | Launchpad UI | `onAppsLoaded()` 清空搜索缓存 | 用户体验差 |
| ISSUE-013 | 应用捕获工具 | 注册表 hive 导出未实现 | 导出的 .vapp 包不包含注册表信息 |
| ISSUE-014 | 应用捕获工具 | `CompareSnapshots()` 未比较文件内容 | 捕获不完整 |

### 5.3 低优先级问题（可选修复）

| 问题 ID | 模块 | 问题描述 | 影响 |
|----------|------|----------|------|
| ISSUE-015 | Hook 引擎 | `CheckHookCoverage()` 可能误报 | 不必要的 Hook 重装 |
| ISSUE-016 | VFS 缓存 | `GetHitRate()` 统计可能不准确 | 命中率统计不准确（误差 < 1%） |
