## 2. 端到端流程测试（P0）

**测试流程：** 安装应用 → 捕获 → 生成 .vapp → 安装 .vapp → 启动应用 → 验证便携性

### 步骤 1：安装应用（以 Notepad++ 为例）
1. 下载 Notepad++ 安装包（`tests\fixtures\npp.8.6.4.Installer.x64.exe`）
2. 启动 Hook 引擎（捕获模式）：`set AI_THINAPP_CAPTURE_MODE=1`
3. 运行安装包，按默认设置安装
4. 安装完成后，停止捕获
5. 应用捕获工具生成快照（文件 + 注册表）

### 步骤 2：捕获应用（生成 .vapp 包）
1. 打开应用捕获工具 GUI
2. 选择快照目录（`C:\Snapshots\Notepad++\`）
3. 点击"生成 .vapp 包"
4. 输出文件：`D:\MyApps\Notepad++.vapp`

### 步骤 3：安装 .vapp 包（到不同目录）
1. 将 `Notepad++.vapp` 复制到另一台电脑（或同一台电脑的不同路径）
2. 双击 `Notepad++.vapp`（或拖入 Launchpad UI）
3. Launchpad 解包到 `D:\PortableApps\Notepad++\`

### 步骤 4：启动应用（验证 Hook 引擎工作）
1. 在 Launchpad 中点击 Notepad++ 图标
2. Launchpad 调用 AppManager::LaunchApp()
3. 应用启动，Hook DLL 被注入
4. 验证文件操作被重定向到 `D:\PortableApps\Notepad++\VFS\`

### 步骤 5：验证便携性（跨目录/跨机器）
1. 在 Notepad++ 中打开文件对话框，确认默认路径是 VFS 目录
2. 编辑文件，保存到 `C:\Temp\test.txt`
3. 确认文件实际被保存到 `D:\PortableApps\Notepad++\VFS\C\Temp\test.txt`
4. 关闭应用，复制 `D:\PortableApps\Notepad++\` 到 U 盘
5. 在另一台电脑上运行 `F:\PortableApps\Notepad++\Notepad++.exe`
6. 验证应用功能正常（文件关联、设置、插件）
