## Microsoft Store 提交步骤

### 1. 注册开发者账号

**要求**:
- Microsoft Partner Center 账号
- 个人或企业开发者账号（$19 个人或 $99 企业）
- 有效的信用卡（用于身份验证）

**步骤**:
1. 访问 https://partner.microsoft.com/
2. 点击「注册」
3. 选择账号类型（个人/企业）
4. 填写个人信息/企业信息
5. 支付注册费用
6. 等待验证（1-3 个工作日）

### 2. 创建应用

**步骤**:
1. 登录 Partner Center
2. 点击「创建新应用」
3. 输入应用名称：`AI ThinApp Portable Launchpad`
4. 保留应用名称
5. 开始提交流程

### 3. 准备提交材料

#### 必填项

| 项目 | 要求 | 状态 |
|------|------|------|
| 应用名称 | 唯一，不超过 256 字符 | ⏳ 待填写 |
| 描述（简短） | 10-200 字符 | ⏳ 待填写 |
| 描述（详细） | 100-2000 字符 | ⏳ 待填写 |
| 截图 | 至少 1 张，最多 8 张 | ⏳ 待准备 |
| 图标 | 300x300 PNG（透明背景） | ⏳ 待准备 |
| 隐私政策 URL | 有效 URL | ⏳ 待创建 |

#### 可选项

| 项目 | 说明 | 状态 |
|------|------|------|
| 功能和支持 URL | 官方网站、支持页面 | ⏳ 待填写 |
| 应用内购买 | 如果需要收费或订阅 | ⏳ 待配置 |
| 系统要求 | Windows 10 版本 1809+ | ✅ 已确定 |
| 分类和子分类 | 工具/系统工具 | ⏳ 待选择 |

### 4. 上传包

**要求**:
- 应用包格式：`.appx` 或 `.msix`（推荐）或 `.msi` 或 `.exe`
- 架构：x64（必需）、x86（可选）
- 签名：必须使用有效的代码签名证书

**步骤**:
1. 在「包」部分点击「选择包文件」
2. 上传 `launchpad-v0.1.0-store.msix`（需先转换）
3. 等待上传完成（进度条显示）
4. 验证包完整性

**注意**: 当前输出为 `.exe` 安装包，需要转换为 MSIX 格式或使用旁加载（企业应用）。

### 5. 填写属性和说明

#### 描述（简短）
```
轻量级便携式应用启动和管理平台，支持环境隔离。
```

#### 描述（详细）
```
AI ThinApp Portable Launchpad Platform 是一个轻量级的应用程序启动和管理平台，支持便携式应用的无缝运行和环境隔离。

主要特性：
• 应用程序启动和管理 - 提供简洁的 GUI 界面
• 环境隔离（Hook 引擎）- 实现基础的 API Hook 机制
• 配置管理 - JSON 格式的配置文件解析和应用管理
• 进程管理 - 启动、监控和终止目标进程

优势：
• 便携性 - 无需安装，可运行在 U 盘或移动设备
• 轻量级 - 资源占用少，不影响系统性能
• 兼容性 - 支持 Windows 10/11 x64

适用场景：
• IT 管理员批量部署便携式应用
• 开发者测试应用在不同环境下的运行
• 普通用户管理常用便携式工具

立即下载并体验！
```

### 6. 提交认证

**步骤**:
1. 检查所有必填项是否完成
2. 点击「提交到 Microsoft Store」
3. 等待认证（通常需要 1-3 个工作日）
4. 查看认证结果

**认证失败常见原因**:
- 应用崩溃或无法启动
- 描述与实际功能不符
- 隐私政策缺失或无效
- 应用包含恶意代码
- 违反 Microsoft Store 政策

## Steam 提交步骤

### 1. 注册 SteamWorks 账号

**要求**:
- Steam 账号
- 绑定 Steam Guard
- 支付 $100 USD 注册费（每个应用）

**步骤**:
1. 访问 https://partner.steampowered.com/
2. 登录 Steam 账号
3. 完成身份验证
4. 支付注册费

### 2. 创建应用

**步骤**:
1. 登录 SteamWorks
2. 点击「创建新应用」
3. 输入应用名称：`AI ThinApp Portable Launchpad`
4. 选择应用类型：「软件」或「工具」
5. 完成创建

### 3. 准备提交材料

#### 必填项

| 项目 | 要求 | 状态 |
|------|------|------|
| 应用名称 | 唯一，不超过 128 字符 | ⏳ 待填写 |
| 简短描述 | 不超过 255 字符 | ⏳ 待填写 |
| 详细描述 | 支持 HTML 格式 | ⏳ 待填写 |
| 截图 | 至少 2 张，最多 20 张 | ⏳ 待准备 |
| 宣传图 | 16:9 比例，1920x1080 | ⏳ 待准备 |
| 图标 | 512x512 PNG | ⏳ 待准备 |
| 小型宣传图 | 230x377 JPG/PNG | ⏳ 待准备 |

#### 可选项

| 项目 | 说明 | 状态 |
|------|------|------|
| 宣传视频 | YouTube 或 Vimeo 链接 | ⏳ 待上传 |
| 系统要求 | 最低配置、推荐配置 | ⏳ 待填写 |
| 定价 | 免费或付费 | ⏳ 待设置 |
| 成就 | 如果支持成就系统 | ⏳ 待配置 |
| 云存档 | 如果支持云存档 | ⏳ 待配置 |

### 4. 上传构建

**要求**:
- 构建格式：`.exe` 安装包或 `.zip` 便携版
- 架构：x64
- 签名：建议使用有效的代码签名证书

**步骤**:
1. 在「技术工具」→「上传」中上传构建
2. 等待上传完成
3. 设置构建为「默认分支」

### 6. 设置定价

**选项**:
- 免费（推荐用于 MVP 阶段）
- 付费（一次性购买）
- 免费试玩（如果有付费版本）

**步骤**:
1. 在「定价」部分选择「免费」
2. 或者设置价格（例如 $4.99）
3. 配置试玩时长（如果适用）

### 7. 提交审核

**步骤**:
1. 检查所有必填项是否完成
2. 点击「准备提交」
3. 等待审核（通常需要 3-5 个工作日）
4. 查看审核结果

**审核失败常见原因**:
- 应用崩溃或无法启动
- 描述与实际功能不符
- 缺少必要的法律文档
- 应用包含恶意代码
- 违反 Steam 分发协议

## 隐私政策

### 要求

**必须包含**:
- 收集的数据类型（如果收集）
- 数据使用目的
- 数据分享对象（如果有）
- 用户权利（访问、更正、删除）
- 联系方式
- 生效日期

**模板**:

```markdown
# 隐私政策

**生效日期**: 2026-05-23

## 1. 引言

AI ThinApp Portable Launchpad Platform（以下简称"本应用"）由 AI ThinApp Team（以下简称"我们"）开发和运营。本隐私政策说明我们如何收集、使用和保护您的信息。

## 2. 信息收集

### 2.1 我们不收集个人信息

本应用**不收集**任何个人信息，包括：
- 姓名、邮箱、电话
- 地理位置
- 设备标识符
- 使用习惯

### 2.2 本地数据

本应用仅在本地存储以下数据：
- 应用配置文件（`config.json`）
- 日志文件（`logs/`）

这些数据**不会**被上传到任何服务器。

## 3. 数据使用

本地数据仅用于：
- 保存您的应用配置
- 调试和故障排除

## 4. 数据分享

我们**不会**与任何第三方分享您的数据。

## 5. 数据安全

我们采取合理措施保护您的数据：
- 数据仅存储在本地
- 不连接网络
- 不传输任何数据

## 6. 您的权利

您有权：
- 访问您的数据（查看配置文件和日志）
- 删除您的数据（删除配置文件和日志）
- 修改您的数据（编辑配置文件）

## 7. 联系我们

如果您对本隐私政策有任何疑问，请通过以下方式联系我们：
- 邮箱：privacy@thinapp.example.com
- 网站：https://example.com/thinapp/privacy

## 8. 政策更新

我们可能会更新本隐私政策。更新后的版本将发布在我们的网站上，并注明生效日期。

## 9. 适用法律的
```
