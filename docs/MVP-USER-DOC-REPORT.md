## 5. 技术细节

### 5.1 文件编码

所有文件使用 **UTF-8 编码（无 BOM）**，Windows 平台 **CRLF 换行符**。

### 5.2 写入方式

所有文件通过 `qclaw-text-file` skill 的 `scripts/write_file.py` 脚本写入，符合跨平台文本文件写入规范。

### 5.3 文档语言

简体中文（zh-CN），面向：
- **用户手册/快速开始/FAQ**：C 端个人用户（避免技术术语）
- **部署文档**：企业 IT 管理员（包含 PowerShell 脚本和 Group Policy）
- **视频教程脚本**：面向视频制作团队
