# AI ThinApp Portable Launchpad Platform - 交互原型说明

## 📋 原型概述

本原型是 **AI ThinApp Portable Launchpad Platform** 项目的 Week 2 交付物，用于演示产品的核心交互流程和设计理念。原型采用纯 HTML + CSS + JavaScript 实现，无需后端或框架支持，可直接在浏览器中打开运行。

### 项目目标

让用户能够：
- 理解"应用便携化"和"沙箱隔离"的核心价值
- 直观地看到沙箱视觉标识的效果
- 体验从添加应用到启动应用的全流程

## 🚀 使用方法

### 直接打开

1. 进入 `assets/prototype/` 目录
2. 双击 `index.html` 即可在浏览器中打开
3. 通过点击页面上的按钮和链接在不同页面间导航

### 推荐浏览器

- **Microsoft Edge**（推荐，Chrome 内核）
- **Google Chrome**
- **Firefox**

> **注意**：由于使用了 Tailwind CSS CDN 和 Font Awesome CDN，需要保持网络连接。

## ⚠️ 已知限制

### 功能限制

1. **无后端支持**：所有数据都是静态模拟，无法真实添加、删除或启动应用
2. **无真实 Hook**：无法演示真实的文件系统/注册表重定向
3. **无真实沙箱**：沙箱效果仅通过 CSS 视觉模拟，无实际隔离能力
4. **安装流程模拟**：商店的"安装"按钮仅演示 UI 流程，不会真实下载或安装应用

### 技术限制

1. **CDN 依赖**：需要网络连接以加载 Tailwind CSS 和 Font Awesome
2. **无数据持久化**：刷新页面后所有状态会重置
3. **浏览器兼容性**：推荐使用现代 Chrome 内核浏览器

## 🎨 设计风格

### 视觉语言

- **设计系统**：参考 Windows 11 Fluent Design 和 Tailwind CSS 默认风格
- **主色调**：紫色（#667eea）- 代表科技感和创新
- **辅助色**：
  - 绿色（#10b981）- 成功、启动、沙箱激活
  - 红色（#ef4444）- 危险操作、删除
  - 蓝色（#3b82f6）- 普通操作、信息展示
- **字体**：Segoe UI（Windows 系统字体）

### 交互设计

- **悬停效果**：卡片向上移动 + 阴影增强
- **按钮状态**：不同颜色区分操作优先级（绿色=主操作，红色=危险操作，灰色=次要操作）
- **过渡动画**：所有交互都有 0.2-0.3s 的平滑过渡
- **反馈机制**：开关切换、按钮点击都有视觉反馈
