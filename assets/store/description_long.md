## 🚀 产品简介

**AI ThinApp Portable Launchpad Platform** 是一款创新的便携应用管理工具。它将您常用的便携式软件集中管理，通过精美的 Launchpad 启动面板一键启动，无需安装，U 盘随身携带。

**核心理念：** 让软件像 Apps 一样随取随用，不污染系统，不留下痕迹。

---

## ✨ 核心功能

### 1. 便携化应用管理

- 📦 **一键添加** - 将便携软件（Portable Apps）添加到启动面板
- 📂 **分类管理** - 按类别整理应用（办公、开发、设计、游戏...）
- 🔍 **快速搜索** - 秒搜应用，无需鼠标点击
- 📝 **自定义图标** - 为每个应用设置个性化图标

### 2. Launchpad 启动面板

- 🎨 **精美界面** - macOS 风格 Launchpad，动画流畅
- ⌨️ **键盘快捷键** - 全局快捷键一键呼出
- 🖱️ **鼠标手势** - 支持手势操作（可选）
- 🎭 **主题切换** - 深色/浅色主题，自动跟随系统

### 3. 沙箱隔离运行

- 🔒 **虚拟化运行** - 应用在沙箱内运行，不修改系统注册表
- 🛡️ **隔离保护** - 应用崩溃不影响系统稳定性
- 🧹 **无痕退出** - 退出后自动清理临时文件
- 🔐 **权限控制** - 可配置应用的文件/网络访问权限

### 4. 一键启动

- ⚡ **极速启动** - 点击即开，无需安装等待
- 📌 **常用置顶** - 常用应用自动置顶显示
- 📊 **使用统计** - 记录应用使用频率
- 🔄 **最近使用** - 智能显示最近使用的应用

### 5. 开源免费

- 💝 **完全免费** - MVP 版本所有功能免费
- 🌍 **开源共享** - 代码公开，透明可信
- 🤝 **社区驱动** - 欢迎贡献，共同完善
- 📖 **文档齐全** - 详细文档，快速上手

## 📥 安装说明

### 方式一：Microsoft Store (推荐)

1. 打开 Microsoft Store
2. 搜索 "AI ThinApp Portable Launchpad"
3. 点击"获取"安装
4. 自动更新，无需手动操作

### 方式二：直接下载 (Itch.io / 独立站)

1. 下载 `AI-ThinApp-Portable-Launchpad.zip`
2. 解压到任意目录（建议 U 盘）
3. 运行 `Launchpad.exe`
4. 无需安装，即刻使用

## 📖 快速上手

### 1. 添加便携应用

1. 点击"+"按钮
2. 选择便携软件可执行文件 (.exe)
3. 自定义图标和名称（可选）
4. 点击"添加"

### 2. 启动应用

- **鼠标：** 点击应用图标
- **键盘：** 全局快捷键 `Ctrl + Space` 呼出 Launchpad，输入应用名搜索

### 3. 沙箱设置

1. 进入"设置" → "沙箱"
2. 启用"沙箱隔离运行"
3. 配置权限（文件访问、网络访问）
